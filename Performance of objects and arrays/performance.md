# Performance of objects and arrays

## Big O of object methods

- Object.keys – O(N)
- Object.values – O(N)
- Object.entries – O(N)
- Object.hasOwnProperty – O(1)

## Big O of array operations

- push – O(1)
- pop – O(1)
- shift – O(N)
- unshift – O(N)
- concat – O(N)
- slice – O(N)
- splice – O(N)
- sort – O(N \* logN)
- forEach/map/filter/reduce/etc. – O(N)
